# SQL

Scripts SQL utilizados para el proyecto de análisis de datos de una clínica dental.

## Contenido

- Creación de tablas de dimensiones y hechos.
- Construcción del modelo estrella.
- Transformaciones ETL.
- Creación de vistas para KPIs.
- Consultas analíticas para Power BI.

## Tecnologías

- PostgreSQL
- DBeaver

## Objetivo

Transformar los datos extraídos de VeviClinic en un modelo analítico optimizado para reporting y visualización.
